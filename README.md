# Marketplace
Teste para admissão na Liferay

Solução para o teste admissional na Liferay para o cálculo de impostos sobre a venda de produtos.

# Tecnologias utilizadas:

- Spring Boot 2.2.5
- Spring JPA
- Thymeleaf
- Bootstrap 3.3.6
- H2 database
- Maven

Projeto realizado no IDE Spring Tool Suite 4.5.1
